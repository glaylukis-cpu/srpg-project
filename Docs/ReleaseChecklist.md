# Release Checklist — Steam 短編 2D SRPG

Steam向けリリース前に確認すべきチェックリスト。
各項目は「☐」から「✅」へ書き換える。

---

## 1. 起動確認

- [ ] Windows 10/11 で正常に起動する
- [ ] Steam外（スタンドアロン）でも起動する
- [ ] 起動後の最初の画面がタイトル画面である
- [ ] FPSが安定している（60 FPS前後）
- [ ] 初回起動時の解像度・ウィンドウモード設定が適切

---

## 2. タイトル画面確認

- [ ] ゲームタイトルが表示されている
- [ ] 「New Game」「Options」「Quit」ボタンが配置されている
- [ ] New Game 選択 → ステージ選択またはステージ1へ遷移
- [ ] Options 選択 → オプション画面へ遷移
- [ ] Quit 選択 → アプリ終了（確認ダイアログ推奨）
- [ ] 背景BGMが再生されている
- [ ] クレジット・バージョン表記が表示されている

---

## 3. ステージ選択確認

- [ ] 全6ステージが一覧表示されている
- [ ] 解放済みステージのみプレイ可能になっている
- [ ] クリア済みステージにバッジやインジケーターがある
- [ ] ステージ名・サブタイトルが正しい
- [ ] キャンセル or Back → タイトル画面へ戻る

---

## 4. Options画面確認

- [ ] Master Volume スライダー動作確認
- [ ] BGM Volume スライダー動作確認
- [ ] SE Volume スライダー動作確認
- [ ] Mute ON/OFF 切替確認
- [ ] フルスクリーン / ウィンドウ切替動作（実装している場合）
- [ ] 設定の保存・再起動後に反映されることを確認

---

## 5. 6ステージ通しプレイ確認

- [ ] ステージ1（チュートリアル）が動作する
- [ ] ステージ2〜5 が正常に進行できる
- [ ] ステージ6（最終面）が正常に完了する
- [ ] 各ステージで敵パターン・地形が正しく配置されている
- [ ] ユニットのステータス・攻撃力・射程が期待通り

---

## 6. 勝利/敗北/リトライ/次ステージ確認

- [ ] 全敵撃破 → Victory表示 → 次ステージまたはALL CLEAR表示へ遷移
- [ ] 自軍全滅 → Defeat画面 → リトライ or タイトルへ戻る
- [ ] ALL CLEAR 演出（最終面クリア時）が正常に表示される
- [ ] Retry で同じステージを再開始できる
- [ ] ステージ間移行時のフェード・トランジションが適切

---

## 7. 操作説明確認

- [ ] タイトル画面またはチュートリアルで操作方法が表示されている
- [ ] キーボード（矢印キー / WASD + Enter）での操作が動作する
- [ ] マウス対応（実装している場合）でユニット選択・移動・攻撃できる
- [ ] 移動範囲・射程表示が正しくハイライトされている

---

## 8. 音量/BGM/SE確認

- [ ] Master Volume 0% で全音声が静かに聞こえない状態になる
- [ ] BGM Volume 調整でBGMのみ減る
- [ ] SE Volume 調整でSEのみ減る
- [ ] 各ステージでBGMが正しく切り替わる
- [ ] 攻撃時のSE、被ダメージSE、VFX音が正しく鳴る
- [ ] 複数音源の同時再生時に破音・割れがない

---

## 9. 解像度確認

- [ ] 1920×1080（Full HD）での描画確認
- [ ] 1280×720（HD）での描画確認
- [ ] ウィンドウモード / フルスクリーンの両方でUIが崩れていない
- [ ] ピクセルアートの拡大率が適切に処理されている
- [ ] 複数モニター環境でも正常に表示される

---

## 10. Debug.Log整理

> 「DebugLogAudit.md」を参照。リリースビルド用のLogger導入後、以下を確認する。

- [ ] PlayerController の選択・ハイライト・移動範囲ログが抑制されている
- [ ] TurnManager の敵AI行動ログが条件付きになっている
- [ ] StageManagerの画面遷移ログが抑制されている
- [ ] AudioManager のmute切替ログが抑制されている
- [ ] BattleUI の操作説明ログが抑制されている
- [ ] Unit.cs の被ダメージ・撃破ログが抑制されている
- [ ] 残すべきログ（Warning/Error/Victory）は引き続き出力される

---

## 11. Windowsビルド確認

- [ ] Unity Player Settings が正しく設定されている（Company/Product/Version）
- [ ] PCビルド（Windows x86_64）が正常に生成される
- [ ] 生成されたEXE がスタンドアロンで起動する
- [ ] ゲームアイコンが正しい
- [ ] ビルド直後のデータ消去・再実行で問題がない

---

## 12. Steam提出前確認

- [ ] Steamworksでアプリ・Depot設定を確認している
- [ ] アプリIDとビルドの対応が取れている
- [ ] スクリーンショット・キャプチャー用画像を準備している
- [ ] Steam Store Page の概要・説明が最終版になっている
- [ ] ビルドファイルをSteamPipeでアップロードできることを確認
- [ ] BetaテストまたはValveレビュー前の内部検証完了
