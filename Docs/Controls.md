# コントロール

## 戦闘中のマウス操作
- **クリック**:
  - ユニットをクリックして選択（味方ユニットのみ）
  - 空のタイルをクリックして選択したユニットを移動
  - 敵ユニットをクリックして攻撃

## 戦闘中のキーボード操作
- **W**: 選択中の味方ユニットを待機させて行動完了
- **U**: 移動後、攻撃や待機で行動確定する前なら移動を取り消し
- **Space**: 敵の脅威範囲表示を切り替え
- **R**: 戦闘中に現在ステージをリトライ
- **Escape または S**: 戦闘中にステージ選択画面を開く

## タイトル画面
- **Up Arrow / Down Arrow**: メニューで項目を選択
- **Enter**: 項目決定

## ステージ選択画面
- **Up Arrow / Down Arrow**: ステージを選択
- **Enter**: 選択したステージを開始
- **Escape/Backspace**: タイトル画面に戻る
- **1〜6の数字キー**: 直接該当するステージを開始

## オプション画面
- **Up Arrow / Down Arrow**: オプションで項目を選択
- **Enter**: 項目決定/切替
- **Escape**: タイトル画面に戻る
- **Left / Right**: 音量や解像度などを調整

## 勝利/敗北後の操作
- **勝利**: 勝利時にバトルログに「VICTORY」が表示される。その後**Enter**を押すと次ステージへ進む。
- **敗北**: 敗北時にバトルログに「DEFEAT」が表示される。その後**Enter**を押すと現在のステージを再読み込み。
